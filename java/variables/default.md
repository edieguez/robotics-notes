# Default values

> This only applies for class and instance variables. **Object references also includes [arrays](arrays.md)**

| Type                           | Default value  |
|:------------------------------:|:--------------:|
| `boolean`                      | `false`        |
| `byte`, `short`, `int`, `long` | 0              |
| `float`, `double`              | 0.0            |
| `char`                         | '\u0000' (NUL) |
| Object references              | `null`         |
