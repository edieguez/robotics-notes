# [JSON path](https://goessner.net/articles/JsonPath/index.html#e2)

[JSON path online evaluator](https://jsonpath.com)

## Elements

| Symbol      | Description                        |
|-------------|------------------------------------|
| `$`         | Root node/element                  |
| `@`         | Current node/element               |
| `.` or `[]` | Child operator                     |
| `..`        | Go up one node/element             |
| `*`         | Wildcard. Matches any node/element |
| `?()`       | Filter operation                   |
