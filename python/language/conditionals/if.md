# Conditional: `if`

## Syntax

### `if`

```python
if boolean_expression:
    pass
```

### `if-else`

```python
if boolean_expression:
    pass
else:
    pass
```

### `if-elif`

```python
if boolean_expression:
    pass
elif another_boolean_expression:
    pass
```

### `if-elif-else`

```python
if boolean_expression:
    pass
elif another_boolean_expression:
    pass
else:
    pass
```
