# Colors in HTML

## [Named](https://developer.mozilla.org/en-US/docs/Web/CSS/Reference/Values/named-color)

```css
color: red;
```

## [Hexadecimal](https://developer.mozilla.org/en-US/docs/Web/CSS/Reference/Values/hex-color)

> Hexadecimal color codes start with a `#` followed by six hexadecimal digits, representing the red, green, and blue components of the color.

```css
color: #FF0000;
```

## [RGB and RGBA](https://developer.mozilla.org/en-US/docs/Web/CSS/Reference/Values/color_value/rgb)

> RGB stands for Red, Green, and Blue
> RGBA stands for Red, Green, Blue, and Alpha (opacity)

```css
color: rgb(255, 0, 0);
color: rgba(255, 0, 0, 0.5); /* 50% opacity */
```

## [HSL and HSLA](https://developer.mozilla.org/en-US/docs/Web/CSS/Reference/Values/color_value/hsl)

> HSL stands for Hue, Saturation, and Lightness
> HSLA stands for Hue, Saturation, Lightness, and Alpha (opacity)

```css
color: hsl(0, 100%, 50%);
```

## [HWB](https://developer.mozilla.org/en-US/docs/Web/CSS/Reference/Values/color_value/hwb)

> HWB stands for Hue, Whiteness, and Blackness

```css
color: hwb(0, 0%, 0%);
```
